Number System Calculator

Single Number Conversion

- Enter a number in the input field.
- Select the base of the input number from the first dropdown.
- Select the desired output number system from the second dropdown.
- Click the "Convert" button to see the converted result.

Arithmetic Operations

- Enter the first number in the input field and select its base from the dropdown.
- Enter the second number in the next input field and select its base from the dropdown.
- Select the arithmetic operation from the dropdown.
- Select the desired output number system from the dropdown.
- Click the "Calculate" button to see the result.
- Click the "C" button to clear all inputs and the result.
